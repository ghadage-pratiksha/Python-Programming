# Basic Programs in Python

This folder contains basic Python programs to understand fundamental concepts of Python programming.

## Programs Included

### 1. hello_world.py
- Simple Python program to print "Hello, World!".
- Introduction to the print() function.

### 2. variables.py
- Demonstrates how to create and use variables in Python.
- Shows storing different types of values.

### 3. data_types.py
- Demonstrates basic Python data types:
  - Integer
  - Float
  - String
  - Boolean

## Language Used

- Python

Run any Python file using:

python filename.py
