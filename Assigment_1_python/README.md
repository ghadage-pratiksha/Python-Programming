# Python Programming - Assignment 1

## Overview

This assignment contains beginner-friendly Python programs that focus on developing logical thinking and problem-solving skills. The programs cover fundamental Python concepts and basic programming techniques.

## Concepts Covered

* Variables
* User Input and Output
* Arithmetic Operators
* Functions
* Return Values
* Conditional Statements (`if-else`)
* `for` Loop
* `while` Loop
* Boolean Values
* Pattern Printing
* Number Operations

## Learning Outcomes

After completing this assignment, I am able to:

* Write and call functions.
* Take input from the user and display output.
* Perform arithmetic operations.
* Use conditional statements for decision making.
* Implement `for` and `while` loops.
* Solve basic logic-building problems.
* Create simple pattern printing programs.
