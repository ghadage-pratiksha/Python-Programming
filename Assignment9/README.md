# Python Programming - Assignment 9

## 📖 Description

This repository contains Python solutions for Assignment 9. The programs focus on pattern printing, currency conversion, factorial-based calculations, and arithmetic operations. These exercises help strengthen logic-building skills through functions, loops, and mathematical problem-solving techniques.

## ✨ Features

- Function-Based Programs
- User Input Handling
- Pattern Printing
- Currency Conversion (USD to INR)
- Even Factorial Calculation
- Odd Factorial Calculation
- Difference Between Even and Odd Factorials
- Time Complexity Analysis

## 📂 Programs

| Question | Description |
|----------|-------------|
| Q1 | Display a pattern using `*` and `#` symbols |
| Q2 | Convert US Dollars (USD) to Indian Rupees (INR) |
| Q3 | Calculate the even factorial of a number |
| Q4 | Calculate the odd factorial of a number |
| Q5 | Find the difference between even factorial and odd factorial |
