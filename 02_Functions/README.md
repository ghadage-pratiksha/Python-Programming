# Functions in Python

This folder contains Python programs demonstrating different types of functions. These examples help understand how functions work using parameters and return values.

## Programs Included

### 1. normal.py
- Simple addition program without using a user-defined function.

### 2. function_without_parameters.py
- Function without parameters and without return value.

### 3. function_with_parameters.py
- Function with parameters and without return value.

### 4. function_with_return.py
- Function without parameters and with return value.

### 5. function_with_parameters_return.py
- Function with parameters and with return value.

## Concepts Covered

- Function Definition
- Function Call
- Parameters
- Return Value
- Main Function
