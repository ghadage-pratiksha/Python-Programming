# Python Programming - Assignment 3

## 📖 Description

This repository contains Python solutions for Assignment 3.

The programs help practice basic programming concepts such as:

- Functions
- Loops
- Conditional Statements
- Factors of a Number
- Even Numbers
- Character Handling
- Case Conversion
- Vowel Checking

## 📂 Programs

| Question | Program |
|----------|---------|
| Q1 | Print first N even numbers |
| Q2 | Display all factors of a number |
| Q3 | Display even factors of a number |
| Q4 | Convert uppercase to lowercase and lowercase to uppercase |
| Q5 | Check whether a character is a vowel |

## 🛠 Language

- Python 3
