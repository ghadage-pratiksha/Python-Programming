# Python Programming - Assignment 10

## 📖 Description

This repository contains Python solutions for Assignment 10. The programs focus on mathematical calculations and unit conversions using functions. The assignment covers area calculations, distance conversion, temperature conversion, and square unit conversion to strengthen problem-solving and programming fundamentals.

- Mathematical Calculations
- Unit Conversions

- Floating-Point Operations
