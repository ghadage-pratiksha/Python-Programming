# Python Programming - Assignment 8

## 📖 Description

This repository contains Python solutions for Assignment 8. The programs focus on conditional statements, loops, number operations, factorial calculation, and multiplication tables. These exercises help improve logical thinking, problem-solving skills, and understanding of fundamental programming concepts.

## ✨ Features

- Function-Based Programs
- User Input Handling
- Conditional Statements
- Loops
- Number Classification
- Digit to Word Conversion
- Factorial Calculation
- Multiplication Tables
- Reverse Table Generation
- Time Complexity Analysis

## 📂 Programs

| Question | Description |
|----------|-------------|
| Q1 | Classify a number as Small, Medium, or Large |
| Q2 | Convert a single-digit number into its word representation |
| Q3 | Calculate the factorial of a number |
| Q4 | Display the multiplication table of a number |
| Q5 | Display the multiplication table in reverse order |

## 🛠 Technology

- Python 3

```bash

```

Example:

python Q1_Number.py
