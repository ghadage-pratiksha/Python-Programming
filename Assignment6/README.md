# Python Programming - Assignment 6

## 📖 Description

This repository contains Python solutions for Assignment 6. The programs are designed to strengthen fundamental programming concepts through practical problem-solving exercises. The assignment covers string handling, user input, conditional statements, arithmetic operations, comparison of values, and percentage calculation using functions.

## ✨ Features

- User Input Handling
- Function-Based Programs
- Conditional Statements
- String Handling
- Number Comparison
- Arithmetic Operations
- Percentage Calculation
- Time Complexity Analysis

## 📂 Programs

| Question | Description |
|----------|-------------|
| Q1 | Accept and display a full name |
| Q2 | Check whether a number is greater than 100 |
| Q3 | Check whether two numbers are equal |
| Q4 | Calculate the multiplication of three numbers |
| Q5 | Calculate the percentage from total and obtained marks |
