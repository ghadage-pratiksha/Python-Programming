# Python Programming - Assignment 7

## 📖 Description

This repository contains Python solutions for Assignment 7. The programs focus on pattern printing, number sequences, loops, and arithmetic operations. These exercises help improve logic-building skills by using functions, iterative statements, and simple mathematical concepts.

## ✨ Features

- Function-Based Programs
- User Input Handling
- Pattern Printing
- Number Sequences
- Loops (for and while)
- Multiples of a Number
- Odd Number Generation
- Time Complexity Analysis

## 📂 Programs

| Question | Description |
|----------|-------------|
| Q1 | Print `$` and `*` pattern based on the given number |
| Q2 | Print numbers from 1 to N |
| Q3 | Print numbers from `-N` to `N` |
| Q4 | Print all odd numbers up to N |
| Q5 | Print the first five multiples of a given number |

## 🛠 Technology

- Python 3

## ▶️ Running the Programs

```bash
python filename.py
```

Example:

```bash
python Q1_Pattern.py
```
